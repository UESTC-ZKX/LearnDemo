# 感知机演示页优化实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在不破坏现有主页面结构的前提下，为感知机章节补充平面分类轻讲解入口，并支持通过新窗口打开高交互独立 Demo 页。

**Architecture:** 复用当前 React + Vite 单入口工程，通过查询参数分流渲染主页面或感知机独立演示页。主页面只改造 `TechnicalEvolutionLabs` 中的感知机实验卡片，独立页拆分为画布、控制区和指标区，分别承载线性可分训练、XOR 单层失败、XOR 两层成功三种场景。

**Tech Stack:** React 19、TypeScript、Vite、Tailwind CSS、Vitest、Testing Library、SVG 可视化
