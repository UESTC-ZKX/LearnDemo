# 反向传播演示页优化实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为反向传播章节补充主页面轻讲解入口，并新增独立教学页，用可交互方式同时讲解误差回传和参数更新。

**Architecture:** 复用当前单入口 React + Vite 工程，通过查询参数分流渲染主页面或独立反向传播教学页。主页面只改造 `TechnicalEvolutionLabs` 中的反向传播区域，独立页拆分为网络画布、控制面板、指标讲解区和时间线/趋势区，支持网络规模切换、单样本模式和按轮模式。

**Tech Stack:** React 19、TypeScript、Vite、Tailwind CSS、Vitest、Testing Library、SVG 可视化
