目标：
实现一个“技术分享前端页面”，主题是：
《从大模型发展，到 Agent 框架选型，再到 Claude 工程实现解析》

要求：
这是一个用于技术分享的单页前端应用，强调“可视化讲解 + 分章节浏览 + 局部交互演示”。
页面不是营销官网，而是适合现场分享/屏幕投影的讲解型产品。

技术栈要求：
- React + TypeScript
- Vite 或 Next.js 均可，优先选择实现成本低、开发快的方案
- Tailwind CSS
- Framer Motion 用于轻量动画
- 图表/流程图优先使用 SVG、CSS、原生组件实现，避免重量依赖
- 代码结构清晰，可继续扩展

产品形态：
- 单页应用（SPA）或单路由多 section
- 左侧为章节导航，右侧为主内容区
- 支持以下 3 大章节：
  1. 大模型发展
  2. Agent 框架选型
  3. Claude 源码/工程实现解析

每个章节必须包含：
- 一个“总览”页面
- 多个“分阶段详细描述”页面
- 每个阶段配一个适合讲解的前端可视化模块

设计目标：
- 深色背景，适合投影
- 信息层次明确，避免花哨
- 强调时间线、流程图、状态流转、模块关系
- 适合技术观众观看
- 演示感强，但不过度依赖复杂 3D 或重图表库

请按以下顺序执行：
1. 先搭项目骨架与页面布局
2. 再补三大章节的静态结构和占位内容
3. 再实现关键交互演示组件
4. 最后统一样式、动效、键盘导航与演示模式

交付要求：
- 可运行项目
- README 中说明启动方式、目录结构、可扩展点
- 所有演示组件都用清晰的 props 设计，后续可替换内容
- 内容数据尽量抽到独立配置文件中
